## ğŸ’¡ How It Works

The portfolio is structured with several key components:

1. **CSS Architecture**: I organized the design system using CSS variables (`:root`) for colors, shadows, spacing, and transitions
2. **Navigation**: Sticky positioning keeps the header accessible while scrolling, with active link highlighting
3. **Hero Section**: Two-column layout with profile image and introduction, featuring responsive text scaling using `clamp()`
4. **Scroll Animations**: AOS library triggers animations as elements enter the viewport, creating a dynamic scrolling experience
5. **Cards & Panels**: Consistent card design with glassmorphism effect and subtle shadows
6. **Responsive Typography**: Used `clamp()` function to create responsive font sizes that scale smoothly
7. **Interactive Elements**: Hover effects and transitions provide visual feedback for user interactions
